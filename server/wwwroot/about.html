About

This web app allows users to generate their own ID by simply entering their information. The generated ID can be downloaded directly or sent to the user’s Gmail.

The application does not use any SQL database, and no user data is stored at any point. All information is used only for generating the ID.

This project was built to showcase my programming skills and demonstrate my ability to deploy a fully functional web application online.
